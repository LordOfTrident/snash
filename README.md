## Introduction
A [shell](https://en.wikipedia.org/wiki/Unix_shell) for [unix](https://en.wikipedia.org/wiki/Unix) and
[linux](https://en.wikipedia.org/wiki/Linux) systems written in Go.
`snash` stands for `snail shell`

## Quickstart
```sh
$ make
$ ./bin/app
```
